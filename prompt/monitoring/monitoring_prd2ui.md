Please build a real-time Compute Resource Monitoring UI of a Large-Model Training Platform:
1. Framework Stack: Use Vue 3 <script setup> composition API syntax, Tailwind CSS, and Lucide Icons (imported via 'lucide-vue-next').
2. UI Design Paradigm:
    2.1 Maintain the sharp, ultra-clean "Notion UI Style", For details, please refer to the @DESIGN.md document.
    2.2 Building upon the Notion style, the metric dashboards can feature various designs. The UI of metric dashboards should feature a highly polished, dark industrial cyber-aesthetic (using Slate/Zinc dark color themes).
3. Monitoring metrics and monitoring UI design: Refer to @monitoring_prd_en.md.
